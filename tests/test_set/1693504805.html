Total OASDI disabled beneficiaries in Missouri zip 63848 remains the same in 2019

Business

By Metric Media News Service | Aug 26, 2020


There were 70 total OASDI disabled beneficiaries in Missouri zip code 63848 in 2019, which is unchanged from 2018, according to the official Social Security Administration website.

The Old-Age, Survivors, and Disability Insurance (OASDI) program is a program of the Social Security Administration funded primarily through payroll taxes. It was signed into law in 1935 by then-President Franklin D. Roosevelt.

The OASDI program aims to supplement a worker’s lost wages due to retirement, disability or death of a spouse.
